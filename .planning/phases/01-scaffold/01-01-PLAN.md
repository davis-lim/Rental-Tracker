  <action>
Run `npm init -y` in the project root to create package.json.

Then install production dependencies:
```
npm install express better-sqlite3 cors
```

Then install dev dependencies:
```
npm install -D vite @vitejs/plugin-react react react-dom react-router-dom concurrently
```

After install, edit package.json to set these scripts:
```json
{
  "scripts": {
    "dev:server": "node server/index.js",
    "dev:client": "vite",
    "start": "concurrently \"npm run dev:server\" \"npm run dev:client\"",
    "build": "vite build"
  },
  "type": "module"
}
```

Set `"type": "module"` at the top level so the project uses ES modules.

Also set `"name": "dads-rental-tracker"` and `"private": true`.

  <action>
Create `vite.config.js` at project root with this exact content:

```js
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';

export default defineConfig({
  plugins: [react()],
  server: {
    port: 5173,
    proxy: {
      '/api': {
        target: 'http://localhost:3001',
        changeOrigin: true,
      },
    },
  },
});
```

Create `.gitignore` at project root with this exact content:

```
node_modules/
dist/
data/
*.db
.DS_Store
```

The `data/` entry ensures the SQLite DB file at `./data/rental.db` (per DATA-03) is not committed to git. The DB is created at runtime.
