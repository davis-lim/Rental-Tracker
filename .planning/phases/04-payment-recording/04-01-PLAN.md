  <action>
Create server/services/rentPayments.js following the exact pattern of server/services/mortgages.js.

**computeDueDate(deadlineDay, month) — pure helper function (not exported):**
- month is YYYY-MM string (e.g. "2026-03")
- deadlineDay is an integer 1–31
- Clamp to end-of-month to handle tenants with deadline_day 29-31 in short months (e.g. Feb):
  ```js
  const [y, m] = month.split('-').map(Number);
  const daysInMonth = new Date(y, m, 0).getDate();
  const clampedDay = Math.min(deadlineDay, daysInMonth);
  return `${month}-${String(clampedDay).padStart(2, '0')}`;
  ```
- This also handles leap years correctly (Feb 29 valid in leap years only)

**computeIsOnTime(paidDate, dueDate, gracePeriodDays) — pure helper (not exported):**
- All three params are strings; gracePeriodDays is an integer
- Compute grace deadline: take dueDate as a Date object, add gracePeriodDays days,
  format back to YYYY-MM-DD string using local date arithmetic:
  ```js
  const due = new Date(dueDate + 'T00:00:00');
  due.setDate(due.getDate() + gracePeriodDays);
  const graceDeadline = due.toISOString().slice(0, 10);
  ```
- Return 1 if paidDate <= graceDeadline (lexicographic string comparison, valid for ISO dates), else 0

**Exported functions:**

`getByTenant(tenantId)` — SELECT all columns from rent_payments WHERE tenant_id = ?, ORDER BY month DESC

`getByMonth(month)` — SELECT rp.*, t.name AS tenant_name, p.address AS property_address
  FROM rent_payments rp
  JOIN tenants t ON t.id = rp.tenant_id
  JOIN properties p ON p.id = t.property_id
  WHERE rp.month = ?
  ORDER BY p.address ASC, t.name ASC

`upsert({ tenant_id, month, paid_date, amount_paid })`:
- Fetch tenant row: `db.prepare('SELECT deadline_day, grace_period_days FROM tenants WHERE id = ?').get(tenant_id)`
- If tenant not found throw Error('Tenant not found')
- Compute dueDate = computeDueDate(tenant.deadline_day, month)
- Compute isOnTime = computeIsOnTime(paid_date, dueDate, tenant.grace_period_days)
- Use INSERT OR REPLACE:
  ```sql
  INSERT OR REPLACE INTO rent_payments (tenant_id, month, paid_date, amount_paid, is_on_time)
  VALUES (?, ?, ?, ?, ?)
  ```
- Return the saved row via: `db.prepare('SELECT * FROM rent_payments WHERE tenant_id = ? AND month = ?').get(tenant_id, month)`

`remove(id)`:
- `db.prepare('DELETE FROM rent_payments WHERE id = ?').run(id)`
- Return `{ deleted: result.changes > 0 }`

Export all four as named exports AND as default object export:
`export default { getByTenant, getByMonth, upsert, remove };`

- Both params required → 400 if missing
