```sql
CREATE TABLE IF NOT EXISTS mortgage_payments (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  mortgage_id INTEGER NOT NULL,
  month TEXT NOT NULL,          -- YYYY-MM
  paid_date TEXT,               -- YYYY-MM-DD
  amount_paid REAL,
  is_on_time INTEGER,           -- 1 = on time, 0 = late
  created_at TEXT DEFAULT (datetime('now')),
  FOREIGN KEY (mortgage_id) REFERENCES mortgages(id),
  UNIQUE(mortgage_id, month)
);
```

Create server/services/mortgagePayments.js following server/services/mortgages.js pattern.

**computeDueDate(dueDay, month) — internal helper (not exported):**
  Same as rent version: return `${month}-${String(dueDay).padStart(2, '0')}`

**computeIsOnTime(paidDate, dueDate) — internal helper (not exported):**
  No grace period for mortgages (MORT-05).
  Return 1 if paidDate <= dueDate (lexicographic ISO date comparison), else 0.

**Exported functions:**

getByMortgage(mortgageId):
  SELECT * FROM mortgage_payments WHERE mortgage_id = ? ORDER BY month DESC

getByMonth(month):
  SELECT mp.*, m.lender AS mortgage_lender, p.address AS property_address
  FROM mortgage_payments mp
  JOIN mortgages m ON m.id = mp.mortgage_id
  JOIN properties p ON p.id = m.property_id
  WHERE mp.month = ?
  ORDER BY p.address ASC, m.lender ASC

upsert({ mortgage_id, month, paid_date, amount_paid }):
  1. Fetch: db.prepare('SELECT due_day FROM mortgages WHERE id = ?').get(mortgage_id)
  2. If not found: throw Error('Mortgage not found')
  3. dueDate = computeDueDate(mortgage.due_day, month)
  4. isOnTime = computeIsOnTime(paid_date, dueDate)
  5. INSERT OR REPLACE INTO mortgage_payments (mortgage_id, month, paid_date, amount_paid, is_on_time)
     VALUES (?, ?, ?, ?, ?)
  6. Return saved row via:
     db.prepare('SELECT * FROM mortgage_payments WHERE mortgage_id = ? AND month = ?').get(mortgage_id, month)

remove(id):
  db.prepare('DELETE FROM mortgage_payments WHERE id = ?').run(id)
  Return { deleted: result.changes > 0 }

Named + default export: export default { getByMortgage, getByMonth, upsert, remove };
