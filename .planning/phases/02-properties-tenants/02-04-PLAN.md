**File 2: src/pages/PropertyDetailPage.jsx**

Replace the placeholder created in 02-02 with the full implementation.

State:
- `property` (object | null)
- `tenants` (array, default [])
- `loading` (boolean, default true)
- `formOpen` (boolean, default false)
- `editingTenant` (object | null)
- `deleteTarget` (object | null)
- `dependentCounts` ({ rent_payments: 0 })

Functions:
- `fetchData()` â€” parallel fetch: GET /api/properties/:id and GET /api/tenants/by-property/:id. Set property and tenants. Set loading false.
- `handleSave(data)` â€” if editingTenant, PUT /api/tenants/:id (body without property_id); else POST /api/tenants (body with property_id). On success: close dialog, fetchData(). On error: set apiError "Something went wrong. Please try again."
- `handleDeleteClick(tenant)` â€” GET /api/tenants/:id/dependents, set dependentCounts, set deleteTarget.
- `handleDeleteConfirm()` â€” DELETE /api/tenants/:deleteTarget.id?confirm=true. On success: close dialog, fetchData(). On error: set apiError "Could not delete. Please try again."
- `handleEditClick(tenant)` â€” set editingTenant, open form dialog.
- `handleAddClick()` â€” set editingTenant to null, open form dialog.

Render (per D-05, D-06, and Layout Structure in UI-SPEC):
